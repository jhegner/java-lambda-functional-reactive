## java-lambda-functional-reactive

### Functional Paradigm

#### Caracteristicas:

* Imutabilidade (estado do objeto nao pode ser modificado)
* First-Class and Higher-Order Functions
* Funções puras (sem efeitos colaterais)
* Referential Transparency

#### Links úteis:
https://www.baeldung.com/java-functional-programming


### Reactive Paradigm

#### Caracteristicas:

* Responsive
* Elastic
* Resilient
* Message Driven

#### Links úteis:

- https://www.reactivemanifesto.org/pt-BR
- https://reactivex.io/
